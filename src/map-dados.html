FRAMEWORK

Os frameworks são ferramentas que buscam facilitar a utilização do usuário e auxiliá-lo no seu processo como profissional de TI. Na área de Dados, os frameworks principais vão de bibliotecas que buscam de simplificar o processamento de dados até plataformas que tornam a manipulação de dados mais robusta, oferecendo estruturas diversas para se trabalhar.
